    Exercicio 2
    Tabela
      Unidade básica de organização numa base de dados
      Consiste em linhas e colunas para representar entidades e atributos

    Query
      Solicitação de dados ou informações numa base de dados
      Geralmente escrita numa linguagem de consulta estruturada (SQL)
      Operações: SELECT, INSERT, UPDATE, DELETE

    Chave Primária
      Identificador único para cada linha numa tabela
      Usado para garantir a integridade dos dados e facilitar a recuperação eficiente de dados

    Índice
        Usado para melhorar a velocidade das operações de recuperação de dados numa tabela
        Fornece acesso rápido a linhas específicas com base em colunas indexadas
